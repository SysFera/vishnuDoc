Chapter 1.  Introduction

Author of the test code : Ibrahima CISSE

Author of the test report : Eugène PAMBA CAPO-CHICHI

Author of the SLURM and LSF tests report : Daouda TRAORE

The tests which follows have been done in the following environment:

  • Database used for TORQUE and SLURM: PostgreSQL 8.4

  • Database used for LSF: mysql 14.14

  • OS used for the TORQUE and SLURM tests: Ubuntu 10.10

  • OS used for the LSF test: CentOS 6.2

  • Cmake version for TORQUE test: V2.6

  • Cmake version for SLURM and LSF tests: V2.8

  • gcc version for TORQUE: V4.3.3

  • gcc version for SLURM: V4.4.5

  • gcc version for LSF: V4.4.6

  • DIET version for TORQUE: V2.6.1

  • DIET version for SLURM and LSF: V2.7

  • Batch schedulers : TORQUE V2.5, SLURM V2.2.1, LSF 7.0.6

  • RAM memory : 4Gb
